## flask-migrate

```bash
pip3 install Flask-Migrate

初始化(只需要执行一次)
flask db init
生成文件
flask db migrate
迁移数据库
flask db upgrade

记得
只需要在 app.py 中导入 models.py 中的类即可。

而且导入全部和导入一个，结果都是可以对所有的表进行创建。

```
